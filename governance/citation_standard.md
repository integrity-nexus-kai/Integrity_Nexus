# Citation Standard

This document defines the citation standard for repositories connected through Integrity Nexus.

---

# Repository Citation

Each connected research repository should include:

- `CITATION.cff`
- author metadata,
- repository URL,
- license information,
- and a stable version or release when available.

---

# Cross-Repository Citation

Integrity Nexus should cite repositories as separate research objects.

It should not collapse authorship, claims, or evidence across repositories.

---

# External Research Citation

When a connected repository relies on external scientific work, that work should be referenced in the repository-specific bibliography.

Integrity Nexus may provide navigation to those bibliographies, but should not become the sole citation source for all projects.

---

# Purpose

Citation discipline preserves traceability, authorship, and scientific credibility.
