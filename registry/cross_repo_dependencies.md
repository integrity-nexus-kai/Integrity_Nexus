# Cross-Repository Dependencies

This document records initial cross-repository dependency relationships inside the Integrity Nexus ecosystem.

The map is intentionally conservative.

It does not claim that one repository proves another.

---

# Structural Dependency Pattern

```text
Shared Concept
  ↓
Domain Projection
  ↓
Repository-Specific Claim
  ↓
Repository-Specific Evidence / Proof / Open Problem
```
