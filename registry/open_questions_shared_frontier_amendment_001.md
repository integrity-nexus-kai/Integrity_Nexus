## Open-Question Position Correction

The canonical Open Questions position is extended.

Open questions are not one homogeneous object class. They must be classified, where applicable, as:

```text
OQ-MAT — Mathematical Open Question
OQ-MET — Methodological Open Question
OQ-EXP — Experimental / Computational Open Question
OQ-SF  — Shared Frontier Question
```

The classes are non-exclusive. A Shared Frontier Question may contain mathematical, methodological, and experimental subquestions.

Question class does not replace any existing canonical status axis or control field.
