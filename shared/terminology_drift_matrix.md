## 1. Purpose

This file prevents shared words from collapsing into identical technical definitions across repositories and scientific domains.

A shared term may provide orientation across the ecosystem, but it does not automatically denote the same object, predicate, structure, claim type, maturity, scope, applicability, lifecycle state, or physical meaning.

This matrix is a governance artifact.

It does not:

- define new theory,
- solve scientific objects,
- create bridges,
- establish physical identity,
- establish Cube ontology,
- close governance questions,
- assign global Completion Readiness,
- report the global current-HEAD synchronization count,

It requires their terminology, scientific domain, status applicability, status axes, Definition State, Bridge State, Relation Class, Relation Target, scope, and transfer boundaries to remain explicit and internally consistent.
