# Purpose

This document prevents duplicate or conflicting ownership of canonical objects across the eight-repository architecture.

Object ownership means:

```text
responsibility for maintaining the canonical repository record
```

not:

```text
proof status or scientific priority
```

# Shared Object Protocol

If an object is used by more than one repository, it must be classified as one of:

```text
imported object
exported object
mirrored summary
candidate interface object
canonical object
```

Only the owning repository may maintain the canonical object unless a transfer is explicitly registered.
