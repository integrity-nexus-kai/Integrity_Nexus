# Source Statement

This document is based on Kai Stefan Dietrich's reported chronology of how the repository structure emerged.

The source chronology can be summarized as:

```text
A long-standing discomfort with GR singularities led to the search for a missing structural component.
That missing component was interpreted as Integrity, not in a moral sense, but as a structural/torsional principle.
Integrity was then tested across systems, including AI alignment and physical theory.
This led to successive repositories, each isolating a different layer of the emerging research architecture.
```
