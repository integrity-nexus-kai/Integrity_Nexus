# Cross-Repository Rule

If a concept appears in multiple repositories, the canonical version must be resolved by:

```text
1. object class,
2. repository source-of-truth responsibility,
3. explicit import/export registration,
4. maturity status,
5. claim-boundary document.
```

No repository may silently promote another repository's candidate object.
